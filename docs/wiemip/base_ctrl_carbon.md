# base-ctrl 1pctCO2 baseline: carbon pools & fluxes

Global totals from the **base-ctrl** 1pctCO2 baseline run (0.5°, 1850–2000).

- **Pools** — VegC, LitC and SoilC — are end-of-year stocks in **Pg C**
  (gridcell `kg C m⁻²` × area, summed globally).
- **Fluxes** — GPP, NPP and Rh — are annual sums of the monthly outputs in
  **Pg C yr⁻¹** (12 months summed per year).

![carbon pools and fluxes](../img/wiemip/base-ctrl/carbon_pools_fluxes.png)

Approximate values over the run (first → last year):

| Variable | Type | First (1850) | Last (2000) |
|----------|------|-------------:|------------:|
| VegC  | pool | 505 Pg C | 524 Pg C |
| LitC  | pool | 174 Pg C | 173 Pg C |
| SoilC | pool | 1596 Pg C | 1595 Pg C |
| GPP   | flux | 107 Pg C yr⁻¹ | 105 Pg C yr⁻¹ |
| NPP   | flux | 49 Pg C yr⁻¹ | 48 Pg C yr⁻¹ |
| Rh    | flux | 49 Pg C yr⁻¹ | 47 Pg C yr⁻¹ |

The pools and fluxes are near-stationary across the baseline period, as
expected for a control run.

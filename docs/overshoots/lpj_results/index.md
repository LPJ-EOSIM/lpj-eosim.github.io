# WIEMIP Overshoots — LPJ Results

LPJ-EOSIM carbon and trace-gas output for the WIEMIP **overshoot** experiment:
a 1000+1000-yr spin-up → historical run (`LPJ-hist`, S2, 1850–2023) and its
control (`LPJ-hist-ctrl`, S0), plus six UKESM1-0-LL-driven overshoot future
scenarios (2024–2300). The forcing data behind these runs is on the
[**Driver Data**](../driver_data.md) page. Older future-scenario figures built
with a previous model version and flag set have moved to the
[**Archive**](../archive.md).

The soil-carbon results and the full investigation of the ~1980 soil-carbon
plateau live on the [**Soil-C plateau recap**](../soilc_plateau_recap.md) page.

This section is split into:

- [**WIEMIP vs regular CRUJRA**](wiemip_vs_crujra.md) — global stocks & fluxes,
  WIEMIP CRUJRA (overshoot, S2) vs regular CRUJRA (S3).
- [**Historical vs control**](hist_vs_control.md) — global carbon & trace-gas
  budgets, the forced historical run vs its drift control, 1850–2023.
- [**Historical + overshoot: stocks & fluxes**](historical_overshoot.md) — the
  full historical run continued by all 6 overshoot scenarios (2024–2300) for
  every key stock and flux, plus maps of the spatial pattern at the end of the
  historical and overshoot periods.
